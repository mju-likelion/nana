# 멋쟁이 사자처럼 홍보 사이트 nana

## Before Start
- 이 프로젝트는 패키지 매니저로 yarn을 권장합니다

## Start Project
```bash
yarn dev
# or
npm run dev

명령어 입력 후
http://localhost:3000에 접속해 보세요
```

```bash
패키지 에러가 발생하면 '사용된 기술'탭에 있는 라이브러리를 설치하세요
yarn add styled-components
yarn add styled-reset
...
```

## Tech Stack
- React, Next.js
- styled-components, Styled-reset
